### Installation:

Just download the code or install it manually:
   
   1. Clone the repo
      ```sh
      git clone https://github.com/Aniol0012/Currency-Converter
      ```
   
   2. Go into the folder
      ```sh
      cd ./Currency-Converter
      ```
    
   3. Execute the script
      ```sh
      ./main.js
      ```
